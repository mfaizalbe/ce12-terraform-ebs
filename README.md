# Terraform EC2 + EBS Practical Task

## Objective
Use Terraform to attach a 1 GB EBS volume to an EC2 instance in the same Availability Zone (AZ) as the EC2’s subnet.

## Requirements
- Create 1 GB EBS volume and attach it to an EC2 instance  
- EBS must be in the same AZ as the EC2 subnet  
- No need to format or mount the EBS volume  
- Submit the **GitHub repo URL** with the working code

## Resources Created
- EC2 instance (Amazon Linux 2023, t3.micro)  
- 1 GB EBS volume attached to the EC2  
- All resources tagged with **`faizal-`** prefix
